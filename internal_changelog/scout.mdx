An experimental AI agent is now available in Scout that can understand natural language questions and automatically generate complex analytics queries. The agent uses semantic search to find relevant report templates and data fields to build its response.

**Pre-requisites**
- Requires `ScoutMastra` feature flag.

#### Why This Matters
This technology aims to make data analysis more accessible by allowing users to get insights without needing to know the underlying data schema or how to use the Query Builder. It significantly lowers the barrier to data exploration.

### ‚úèÔ∏è Edit Scout AI Charts Directly in Report Builder

**What's New**  
When viewing a data chart generated by Scout AI, you can now open it directly in the Analytics Report Builder for further editing and exploration. A new "Edit with report builder" option has been added to the chart's action menu.

**Pre-requisites**
- Requires `AiChat` and `AnalyticsV2` feature flags.

<Accordion title="See More">

#### Where You'll See It
- *AI > Chat* (in the three-dot dropdown menu of a chart generated by Scout).

#### What You Can Do
- After Scout generates a chart, click the menu on the chart card and select "Edit with report builder".
- This action navigates you to the *Analytics > Query Builder* page with the complete query from the Scout-generated chart pre-loaded and ready for modification.

#### Why This Matters
Previously, to refine a chart created by Scout, you had to manually reconstruct the entire query in the Report Builder. This enhancement creates a seamless workflow, allowing you to use Scout for initial discovery and then transition to the full power of the Report Builder with a single click to fine-tune your analysis.
</Accordion>

<Update label="25 November 2025" description="1 Updates">

### üìä Scout AI Query Results Capped at 1,000 Rows

**What's New**  
To improve performance and prevent browser issues, analytics queries generated by the Scout AI assistant are now limited to a maximum of 1,000 rows.

**Pre-requisites**
- Requires `Scout` feature flag
- Requires Admin access

<Accordion title="See More">

#### Where You'll See It
- *Scout AI Assistant* (when asking for analytics data that returns a table)

#### What You Can Do
- If you ask for a query that would return more than 1,000 rows (e.g., "show me all trips from last year"), the results will be automatically capped at 1,000.
- If you ask for a specific number below 1,000 (e.g., "show me the top 10 drivers"), that limit will be respected.

#### Why This Matters
Requesting extremely large datasets could cause the Admin portal to become slow or unresponsive. This limit ensures a stable and performant experience when using the Scout AI for data analysis.
</Accordion>

- View charts with many categories without the page becoming excessively long; simply scroll within the chart's container to see all the data.

#### Why This Matters
These changes improve both the intelligence of the AI assistant and the usability of the charts it generates. You get better answers from Scout, and the resulting visualizations are cleaner and easier to manage.
