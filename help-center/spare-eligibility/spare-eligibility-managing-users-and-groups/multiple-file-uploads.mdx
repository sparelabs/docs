description: "Learn how to enable multiple file uploads in your Case Forms and custom fields for eligibility documentation."
---

<Note>
Multiple file uploads are useful when agents or riders need to submit several documents at once, such as for paratransit eligibility determinations.
</Note>

Multiple file uploads can be easily added to your Eligibility forms through custom fields, allowing for streamlined document collection processes.

## Upload Specifications

<CardGroup cols={3}>
<Card title="Maximum Files" icon="files">
Up to **100 files** per upload
</Card>

<Card title="File Size Limit" icon="file-arrow-up">
**5 MB** maximum per file
</Card>

<Card title="Supported Formats" icon="file-check">
**PNG, JPEG, PDF**
</Card>
</CardGroup>

## How to Add Multiple File Upload Option

<Steps>
<Step title="Access Case Template">
In **Spare Eligibility**, select the **Case Template** you'd like to edit.
</Step>

<Step title="Add or Edit Form">
Either select **"+ Add Form"** to create a new form, or click on an existing form to modify it.

![Screenshot of Case Template interface showing form options](/images/spare-eligibility/managing-users-and-groups/case-template-form-options.png)
</Step>

<Step title="Add Form Field">
In the form template, click **"+ Add Form Field"**.

![Screenshot showing Add Form Field button in form template](/images/spare-eligibility/managing-users-and-groups/add-form-field-button.png)
</Step>

<Step title="Configure Multiple Files Field">
Choose the **"Multiple Files"** custom field option and fill out the required information. Click **"Create Custom Field"** to save your changes.

![Screenshot of Multiple Files custom field configuration](/images/spare-eligibility/managing-users-and-groups/multiple-files-field-config.png)
</Step>
</Steps>

## Rider Experience

Once configured, riders will see an intuitive file upload interface that allows them to select and upload multiple documents simultaneously:

![Screenshot showing the rider-facing multiple file upload interface](/images/spare-eligibility/managing-users-and-groups/multiple-file-upload-interface.png)

<Tip>
The multiple file upload feature streamlines the document submission process for both riders and agents, reducing the time needed to complete eligibility assessments.
</Tip>
