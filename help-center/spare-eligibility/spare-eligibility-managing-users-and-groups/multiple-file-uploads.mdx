---
title: "Multiple File Uploads"
description: "Learn how to enable multiple file uploads in your Case Forms and custom fields for eligibility documentation."
---

<Note>
Multiple file uploads are useful when agents or riders need to submit several documents at once, such as for paratransit eligibility determinations.
</Note>

Multiple file uploads can be easily added to your Eligibility forms through custom fields, allowing for streamlined document collection processes.

## Upload Specifications

<CardGroup cols={3}>
<Card title="Maximum Files" icon="files">
Up to **100 files** per upload
</Card>

<Card title="File Size Limit" icon="file-arrow-up">
**5 MB** maximum per file
</Card>

<Card title="Supported Formats" icon="file-check">
**PNG, JPEG, PDF**
</Card>
</CardGroup>

## How to Add Multiple File Upload Option

<Steps>
<Step title="Access Case Template">
In **Spare Eligibility**, select the **Case Template** you'd like to edit.
</Step>

<Step title="Add or Edit Form">
Either select **"+ Add Form"** to create a new form, or click on an existing form to modify it.

![Screenshot of Case Template interface showing form options](https://spare-74fa64f80617.intercom-attachments-1.com/i/o/674275983/7edf966f1ddbccd443a282aa/yx4-JFssA8GFuAN5zI2_MsBFoM_TOxnf4w.png?expires=1753820100&signature=f4448df9da3db217604c031c30c2cc3f696515760b432736e86739c8e9bd33d7&req=cicjFM57lIlcFb4f3HP0gE63waD9boimzbL02mwYBeVpyNeDDC%2B%2Bq2XxXUrh%0ACVE%3D%0A)
</Step>

<Step title="Add Form Field">
In the form template, click **"+ Add Form Field"**.

![Screenshot showing Add Form Field button in form template](https://spare-74fa64f80617.intercom-attachments-1.com/i/o/674275986/9d3bbe29a670ad61bee24104/8AYtz9CJR6OPbAiz3aG8GSn8Cswo0RAAHg.png?expires=1753820100&signature=db310987c486f0d964c8d7f074e0037f6149c9eaf775007aeaee7213cd0ba6e1&req=cicjFM57lIlZFb4f3HP0gJQZi%2FPD5zdAlyI0ir3mSJGkYRXJgSfJHXUdwya2%0AdZk%3D%0A)
</Step>

<Step title="Configure Multiple Files Field">
Choose the **"Multiple Files"** custom field option and fill out the required information. Click **"Create Custom Field"** to save your changes.

![Screenshot of Multiple Files custom field configuration](https://spare-74fa64f80617.intercom-attachments-1.com/i/o/674275992/8690b7e3c69a76b7d44cc0e8/pKSN8ehS-dxQPJ4YIVfEzvvgLyVoza_5Zw.png?expires=1753820100&signature=e020ab4b3a1040da6d65129f8649006fd100b9e2788d4879e91ee3e28fd12fc1&req=cicjFM57lIhdFb4f3HP0gJjsztVxjhphJH240h8NXEX56CnRxEZm9qjBYBaN%0AK%2Bk%3D%0A)
</Step>
</Steps>
