title: "Case Forms: Uploading Multiple Files"

---

Enable multiple file uploads on your Eligibility forms to streamline document collection and improve the user experience for applicants who need to submit multiple supporting documents.

## Adding a Multiple File Upload Field

<Steps>
  <Step title="Navigate to Case Template">
    Go to **Spare Eligibility** and select the **Case Template** you'd like to edit.
  </Step>
  
  <Step title="Access Form Builder">
    Choose one of the following options:
    - Select **"+ Add Form"** to create a new form
    - Click on an existing form to modify it
    
    ![Case template interface showing form creation options](/images/spare-eligibility/managing-case-templates/form-creation-options.png)
  </Step>
  
  <Step title="Add Form Field">
    In the form template, click **"+ Add Form Field"** to open the field selection menu.
    
    ![Form builder interface with Add Form Field button highlighted](/images/spare-eligibility/managing-case-templates/add-form-field-button.png)
  </Step>
  
  <Step title="Configure Multiple Files Field">
    1. Choose the **"Multiple Files"** custom field from the available options
    2. Fill out the field information including:
       - Field name and description
       - Upload requirements and instructions
       - Any specific file format requirements
    3. Click **"Create Custom Field"** to save your changes
    
    ![Multiple Files field configuration dialog with field settings](/images/spare-eligibility/managing-case-templates/multiple-files-field-config.png)
  </Step>
</Steps>

## Applicant Experience

The multiple file upload field provides a user-friendly interface for applicants to submit their documents:

![Multiple file upload interface showing different upload states for applicants](/images/spare-eligibility/managing-case-templates/multiple-file-upload-states.png)

The interface shows three different upload states:
- **Ready to upload**: Clean interface waiting for file selection
- **Upload in progress**: Progress indicators during file transfer
- **Upload complete**: Confirmation of successful file submissions

## Technical Specifications

<Card title="Upload Limits & Requirements" icon="info-circle">
  <div style={{ display: 'grid', gap: '12px' }}>
    <div>
      <strong>Maximum Files Per Upload:</strong> 100 files
    </div>
    <div>
      <strong>Maximum File Size:</strong> 5 MB per file
    </div>
    <div>
      <strong>Supported Formats:</strong> PNG, JPEG, PDF
    </div>
  </div>
</Card>

<Warning>
Ensure applicants are aware of file size limits and supported formats to prevent upload failures. Consider adding these specifications to your form instructions.
</Warning>

## Best Practices

<Accordion title="Optimization Tips">
  <AccordionItem title="Field Naming">
    - Use descriptive field names that clearly indicate what documents are needed
    - Include examples of acceptable documents in the field description
    - Specify any required document types or formats
  </AccordionItem>
  
  <AccordionItem title="User Instructions">
    - Provide clear instructions about file requirements
    - Include file size and format limitations in the field description
    - Consider adding examples of properly formatted documents
  </AccordionItem>
  
  <AccordionItem title="File Organization">
    - Group related document uploads logically within your form
    - Consider using separate multiple file fields for different document categories
    - Add content sections to explain document requirements between upload fields
  </AccordionItem>
</Accordion>
