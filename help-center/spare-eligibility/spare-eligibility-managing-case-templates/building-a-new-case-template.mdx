To start managing your cases effectively, you'll first need to build a case template that will help you unify the process. Navigate to the **Case Templates** tab where all your case templates are stored and organized.

## Case Template Components

Each case template consists of four essential elements:

<CardGroup cols={2}>
  <Card title="Case Template Details" icon="gear">
    Name, keyword identifier, status, and case timer configuration
  </Card>
  <Card title="Form Templates" icon="file-lines">
    Fillable forms that can be shared with your customers
  </Card>
  <Card title="Letter Templates" icon="envelope">
    General case letters that can be added to the rider's case
  </Card>
  <Card title="Case Statuses" icon="list-check">
    Color-coded case process steps for tracking progress
  </Card>
</CardGroup>

## Creating a New Case Template

<Steps>
  <Step title="Start Building">
    Click **"+ Add Case Template"** to start building from scratch.
    
    <img src="https://spare-74fa64f80617.intercom-attachments-1.com/i/o/964468322/b3a114d5f5d77ace4f2b76f7/OwWNVRwlGYv1ujyZez3amYlpvdqdVc0VoQ.png?expires=1753820100&signature=0d458bf3c91a7862b9ee4c4789f7612372d4872868f64b495e819df0f468197c&req=fSYjEs92noNdFb4f3HP0gFxJkxYsqP7ZX%2BtMK65x%2FG7JG7W5zLEfSEgD0KkK%0ARSo%3D%0A" alt="Add Case Template button in the interface" />
  </Step>
  
  <Step title="Configure Basic Information">
    A new menu will appear. Enter the following details:
    - **Case name**: A descriptive name for your template
    - **Keyword (identifier)**: A unique identifier for the template
    - **Case timer**: Optional time limit configuration
    
    <img src="https://spare-74fa64f80617.intercom-attachments-1.com/i/o/964468330/78934888b04ddb426b40a81d/HK0N--GPwjofh8gRHW_fBAV3myMfFPvoAA.png?expires=1753820100&signature=9178a823a22f5bf1690055c85ae6b12bc05d0039a706a132f90350d35ac69fc0&req=fSYjEs92noJfFb4f3HP0gLLx%2BmqHEjgsDefNKaep8y07jW%2BnGBA1qSTN5t8c%0AZA4%3D%0A" alt="Case template creation form with name and keyword fields" />
  </Step>
  
  <Step title="Set Timer (Optional)">
    To configure a case timer:
    1. Check the **"Enabled"** box
    2. Set the time limit in full days
    
    <Tip>
      Case timers help ensure cases are processed within specific timeframes and can trigger notifications when deadlines approach.
    </Tip>
    
    <img src="https://spare-74fa64f80617.intercom-attachments-1.com/i/o/964468337/eab591453085ef90ad8a61d0/L6Z_jEmrkhDvdw1GYgikmx5qCTcY1mVViA.png?expires=1753820100&signature=dcb40872d967a964f03f8e84d50a72f69fe606df096ae6cec9c09663d8f3b058&req=fSYjEs92noJYFb4f3HP0gGT3PwsSOlPrmuJCHmaBG6yZpyWbKKpLTN%2BI%2FalR%0A4l8%3D%0A" alt="Timer configuration settings with enabled checkbox and days input" />
  </Step>
  
  <Step title="Create Template">
    Click **"Create New Case Template"** to save your changes. You'll be automatically redirected to the case template builder.
  </Step>
</Steps>
