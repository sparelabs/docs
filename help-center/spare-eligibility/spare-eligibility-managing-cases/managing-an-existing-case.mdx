## Changing Case Statuses

Each time a step is completed, you can easily change the case status from the case information page.

<Steps>
  <Step title="Access Status Options">
    Simply click on the current status to open status options

    />
  </Step>
  
  <Step title="Select New Status">
    Choose the appropriate status from the dropdown menu
  </Step>
  
  <Step title="View Updated Status">
    A new status will now be displayed in the Case Type tab

## Generating a Personalized Case Letter

You can easily add a personalized letter using Case Letter Templates.

<Steps>
  <Step title="Access Letter Generation">
    Scroll down to the **Generated Case Letters** section and click **"+ Generate Case Letter"**
  </Step>
  
  <Step title="Select Template">
    You will be redirected to the **"Generate Case Letter"** menu. Select a case letter template applicable to the case type
    
    <img 
      src="https://spare-74fa64f80617.intercom-attachments-1.com/i/o/964502365/5141f0b661a9ada40531802d/P5g94pkOEfs77bjm6Fqu2dxo7A-3xTtl-w.png?expires=1753820100&signature=cf4d257f96a4ce42bf47d7f0bc04f2a950318a08a5d05bf1341f721e1c625cc1&req=fSYjE8l8nodaFb4f3HP0gDhvEX5vSgwIN%2BjltTZTR7WU%2BobxSBsfq30RTk4B%0AwW0%3D%0A"
      alt="Generate Case Letter interface showing template selection and download option"
    />
  </Step>
  
  <Step title="Generate and Download">
    Click **"Generate and Download the letter"** to retrieve the PDF version
  </Step>
</Steps>

<Tip>
Case letters are automatically personalized with the rider's information and case details from your selected template.
</Tip>
