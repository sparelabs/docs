---
title: "Creating New Memberships"
description: "Learn how to assign users to groups by creating new memberships through the Case Page, Rider Page, or Groups Page in Spare Eligibility."
---

Users can be assigned to groups using any of the three methods below. Choose the method that works best for your workflow.

<CardGroup cols={3}>
  <Card title="Case Page" icon="folder" href="#the-case-page">
    Add memberships directly from the case information page
  </Card>
  <Card title="Rider Page" icon="user" href="#the-rider-page">
    Manage memberships from the rider's profile
  </Card>
  <Card title="Groups Page" icon="users" href="#the-groups-page">
    Add members to groups from the groups management page
  </Card>
</CardGroup>

## The Case Page

From the Case Information Page, scroll down to the **"Rider's Group Memberships"** section. All currently active memberships will be displayed there.

![Rider's Group Memberships section showing active memberships](https://spare-74fa64f80617.intercom-attachments-1.com/i/o/964496151/b467472edb9aa1e43c213716/_3z4Cgc_dI-RgRrOivt7Frn5-Pr0y2Pb8g.png?expires=1753820100&signature=2816cf262a3d1d574318cb3a929a030189c03d8de64442c01d30741d50943e28&req=fSYjEsB4nIReFb4f3HP0gH1ZLR%2BAnNzD4QbqeKa2I82o0Xpl%2F9adDyZ8f7Sl%0AOLE%3D%0A)

<Steps>
  <Step title="Add New Membership">
    Click **"+ Add Membership"** to begin creating a new group membership.
  </Step>
  <Step title="Enter Group Details">
    Type in the **group name** you'd like the rider to be added to and **set the expiration date**. If the membership is permanent, select "Forever".
  </Step>
  <Step title="Save Changes">
    Click **"Create New Membership"** to save the changes.
  </Step>
</Steps>

![Add membership dialog with group name and expiration date fields](https://spare-74fa64f80617.intercom-attachments-1.com/i/o/964496159/1fab63ccd9a64e688512e3ae/OttJG2l-huplvnY0QMVt6B-zifOMH9-kHQ.png?expires=1753820100&signature=a9ece287c1800860438e8a7787af5cfcfb69a9d1efa96e7554c069c6390ca96b&req=fSYjEsB4nIRWFb4f3HP0gM9u27j7T8oacmHWmNDXZdEqHZLCX3K6EdeP9wed%0AjeM%3D%0A)

## The Rider Page

From the Riders tab (**Users → Riders**), locate the rider (you can type in their name, ID or phone number) and open their profile. Scroll down to **"Memberships"** and click **"+ Add Membership"**.

![Rider profile page showing the Memberships section with Add Membership button](https://spare-74fa64f80617.intercom-attachments-1.com/i/o/964496170/5d8ff4cb8a569c7aa1ebb5d3/SbasuJlz0wfNzbVYYRjoRkSDSTDspKnN2A.png?expires=1753820100&signature=0f471c1a1ed6b8c9339489e2dd7eb90b91c74323aa5dc238f4921a48181d1e48&req=fSYjEsB4nIZfFb4f3HP0gJOvtGpsnlH6xxxjjEhR%2FQxAc0VT1oWSALGsTulb%0Apk8%3D%0A)

<Steps>
  <Step title="Enter Group Details">
    Type in the **group name** you'd like the rider to be added to and **set the expiration date**. If the membership is permanent, select "Forever".
  </Step>
  <Step title="Save Changes">
    Click **"Create New Membership"** to save the changes.
  </Step>
</Steps>

![Create membership form with group name and expiration settings](https://spare-74fa64f80617.intercom-attachments-1.com/i/o/964496186/798a94187ad8898e41279c6b/OttJG2l-huplvnY0QMVt6B-zifOMH9-kHQ.png?expires=1753820100&signature=790987f8fa19e7abbd8fcdcaa1d4e91f42bc8b640065e295df41e8f4554173bd&req=fSYjEsB4nIlZFb4f3HP0gKpV0xZlOe5ACt60Sg%2BIrBo5dKXiWY65EdZVDBp8%0Am3w%3D%0A)

## The Groups Page

To add a new Membership to a Group, go to **Users → Groups** and select the **Groups** tab.

![Groups navigation showing the Groups tab selection](https://spare-74fa64f80617.intercom-attachments-1.com/i/o/964496197/a026fc011225e46004ecafb7/tMUtnJgLRJtKwBcKno7R-QjJDtD3ayYYWQ.png?expires=1753820100&signature=b5d4c2724b33f7ba0660250332b5d32e9a463985b409abcf11f72bcfbb8c97f8&req=fSYjEsB4nIhYFb4f3HP0gP4PXr0HBbVjvzd3sBneUstFIl%2FSwKd6Hm5gMAwD%0AExs%3D%0A)

A group information page will appear. Click **"+Add Membership"**.

![Group information page with Add Membership button](https://spare-74fa64f80617.intercom-attachments-1.com/i/o/964496205/c4be05d22117970ca0c5a18a/r80U8O7_F_Vv855rz0_t5MACTJweUa3bTw.png?expires=1753820100&signature=d82674ae1747d5b9131a3a91e6429509c717f67dfa2b33d2457ccbd8d8348431&req=fSYjEsB4n4FaFb4f3HP0gCIze5BIdCcn4%2B%2BP4zbyMv4jhYRig5qweMRd4c9a%0ARB0%3D%0A)

<Steps>
  <Step title="Verify Group Information">
    **Check if the group name is correct** (otherwise, type in the correct one), and **type in the Rider name** (select the rider from the search results).
  </Step>
  <Step title="Set Expiration">
    Decide if the membership has an **expiry date**. If it doesn't, keep the "Forever" box checked. If it does, uncheck the box and set the expiry date and time.
  </Step>
  <Step title="Create Membership">
    Click **"Create New Membership"** to apply the changes.
  </Step>
</Steps>

![Add membership form showing group name, rider selection, and expiration settings](https://downloads.intercomcdn.com/i/o/977164511/f1b0ddc52eed65f6a7d6dbd9/Screenshot+2024-02-28+at+3_45_17%E2%80%AFPM.png?expires=1753820100&signature=ebafd24fb57e69f0c6b3bb69f4496af86a76ca85d6bfeed2c0a36471de978691&req=fScgF896mIBeFb4f3HP0gI2uQ%2BdRFcetUTQONiOCsIBv%2B1IyafoB6FoyrnLN%0AXkw%3D%0A)

<Tip>
  All three methods achieve the same result - creating a new group membership for a rider. Choose the method that fits best with your current workflow and the page you're already working on.
</Tip>
