This guide covers the essential steps for creating and managing new cases in Spare Eligibility, including rider search functionality and case assignment workflows.

<Info>
Before creating a case, ensure you have the necessary permissions and that the rider profile exists in the system.
</Info>

## Adding a New Case

<Steps>
  <Step title="Navigate to Cases">
    Go to the **Cases** tab within Eligibility and click **"+ Add Case"**.
  </Step>
  
  <Step title="Select Case Type">
    - When adding a case from the **All Cases** tab, you will be invited to select the case type
    - When adding a case from a **specific case type tab**, the case will automatically be assigned to that case type
  </Step>
</Steps>

![Screenshot of All Cases screen with Add Case button highlighted](/images/spare-eligibility/managing-cases/add-case-button-all-cases.png)

### Searching for Riders

From the **Add Case** form, search for the rider using any of the following:
- Rider name
- Phone number  
- Email address

<Warning>
If you are creating a case for a new rider whose profile is not in the system yet, you will be offered to [create a Rider Profile first](https://help.sparelabs.com/en/articles/8955368-managing-riders).
</Warning>

### Completing the Case Creation

1. Add internal notes if necessary
2. Click **"Create New Case"** to save your changes

![Screenshot of Add Case form showing Add Rider and Create New Case buttons](/images/spare-eligibility/managing-cases/add-case-form-rider-creation.png)

<Tip>
Once the case record is created, you will be able to see it in both the **Case Type** tab and the **All Cases** tab.
</Tip>

![Screenshot of All Cases screen displaying a newly created case record](/images/spare-eligibility/managing-cases/newly-created-case-display.png)

## Assigning Cases

<Steps>
  <Step title="Access Case Information">
    Click on the case record to access its information page and settings.
    
    ![Screenshot of All Cases screen with an example case highlighted for selection](/images/spare-eligibility/managing-cases/case-selection-for-assignment.png)
  </Step>
  
  <Step title="Assign the Case">
    From the case information page, click **"Assign to"** and type in the specialist name.
    
    ![Screenshot of case information page with Assign To button highlighted](/images/spare-eligibility/managing-cases/assign-case-to-specialist.png)
  </Step>
</Steps>

### Changing Case Assignment

To change the assignee, click on the specialist name again and type in the new name.

![Screenshot of case page showing updated assignee information](/images/spare-eligibility/managing-cases/case-assignee-updated.png)

### Viewing Cases by Assignee

<Steps>
  <Step title="Navigate to All Cases">
    Go to **Cases** → **All Cases**
  </Step>
  
  <Step title="Apply Filter">
    Filter the results by **"Assigned"** → **"Assigned to"** → **Employee Name**
  </Step>
  
  <Step title="Apply Changes">
    Click **"Done"** to apply the filter.
  </Step>
</Steps>
