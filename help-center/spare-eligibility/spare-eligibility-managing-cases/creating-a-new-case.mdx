This guide covers the essential steps for creating and managing new cases in Spare Eligibility, including rider search functionality and case assignment workflows.

<Info>
Before creating a case, ensure you have the necessary permissions and that the rider profile exists in the system.
</Info>

## Adding a New Case

<Steps>
  <Step title="Navigate to Cases">
    Go to the **Cases** tab within Eligibility and click **"+ Add Case"**.
  </Step>
  
  <Step title="Select Case Type">
    - When adding a case from the **All Cases** tab, you will be invited to select the case type
    - When adding a case from a **specific case type tab**, the case will automatically be assigned to that case type
  </Step>
</Steps>

### Searching for Riders

From the **Add Case** form, search for the rider using any of the following:
- Rider name
- Phone number  
- Email address

<Warning>
If you are creating a case for a new rider whose profile is not in the system yet, you will be offered to [create a Rider Profile first](https://help.sparelabs.com/en/articles/8955368-managing-riders).
</Warning>

### Completing the Case Creation

1. Add internal notes if necessary
2. Click **"Create New Case"** to save your changes

### Viewing Cases by Assignee

<Steps>
  <Step title="Navigate to All Cases">
    Go to **Cases** → **All Cases**
  </Step>
  
  <Step title="Apply Filter">
    Filter the results by **"Assigned"** → **"Assigned to"** → **Employee Name**
  </Step>
  
  <Step title="Apply Changes">
    Click **"Done"** to apply the filter.
  </Step>
</Steps>
