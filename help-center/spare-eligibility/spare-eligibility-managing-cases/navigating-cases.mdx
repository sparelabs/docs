## Navigating the Cases Dashboard

When you navigate to the Cases tab, you'll be redirected to the **All Cases** section by default.

<Steps>
  <Step title="Navigate between case types">
    Click on the case type name on the left sidebar to switch between different case types.
  </Step>
  <Step title="View case status dashboard">
    Each case type displays a dashboard with case statuses. Click on any segment to sort the cases by that status.
  </Step>
</Steps>

![Cases dashboard showing case type navigation and status segments](/images/spare-eligibility/managing-cases/cases-dashboard-navigation.png)

### Adding New Cases

You can create a new case by clicking the **"+Add Case"** button and selecting a case type. 

<Card title="Learn More" icon="plus" href="http://help.sparelabs.com/en/articles/8955339-creating-a-new-case">
  Creating a New Case
</Card>

![Add Case button interface](/images/spare-eligibility/managing-cases/add-case-button.png)

### Filtering and Searching Cases

To quickly find an existing case, you can filter case records using multiple criteria.

<Callout type="info">
  You can filter your results by:
  - Case status
  - Rider's name
  - Rider's email or phone number
  - Date the case was received
  - Person assigned to the case
</Callout>

<Steps>
  <Step title="Access filters">
    Click the "Filter" button to open the filtering options.
  </Step>
  <Step title="Select search conditions">
    Choose your desired filter criteria from the available options.
  </Step>
</Steps>

![Filter interface showing search options for cases](/images/spare-eligibility/managing-cases/filter-interface.png)

## Navigating the Case Type Home Page

Each case type has its own structure, but the general layout follows a consistent pattern. All active cases of a specific type are displayed on the case type home page (**Cases → Case type name**).

### Available Actions

From the Case Type home page, you can:

<CardGroup cols={2}>
  <Card title="View Case Statuses" icon="eye">
    Move between tabs to sort case records by status
  </Card>
  <Card title="Search Cases" icon="magnifying-glass">
    Filter results by various criteria
  </Card>
  <Card title="View Templates" icon="file-text">
    Access the case type template
  </Card>
  <Card title="Generate PDFs" icon="file-pdf">
    Print or view PDF versions of forms
  </Card>
  <Card title="Share Forms" icon="share">
    Share fillable links to case forms
  </Card>
  <Card title="Add Cases" icon="plus">
    Create new cases for the selected type
  </Card>
</CardGroup>

You can access a specific case record by clicking on the record line.

![Case type home page showing available actions and case list](/images/spare-eligibility/managing-cases/case-type-home-page.png)

### Case Record Information

Each case record on the Case Type home page displays:

- **Case status** - Current state of the case
- **Rider's name** - Name of the applicant
- **Creation date** - When the case record was created
- **Assignee** - Person responsible for the case

## Navigating an Active Case Information Page

When you open a case information page (**Cases → Case type name → Case record**), you'll see several key components:

### Page Components

<Accordion title="Case General Information">
  Includes rider name, case type, case creation date and timer, notes, case status, and case assignee.
</Accordion>

<Accordion title="Case Forms">
  Shows completion state and the date and time forms were last updated or submitted.
</Accordion>

<Accordion title="Case Letters">
  Displays letters generated and sent out to the rider.
</Accordion>

<Accordion title="Rider's Membership Information">
  Contains relevant membership details for the rider.
</Accordion>

<Accordion title="Case Activity Log">
  Chronological record of all actions taken on the case.
</Accordion>

<Callout type="tip">
  Most case elements are interactive. Clicking on them will redirect you to the associated pages or forms.
</Callout>
