title: "Location-Specific Notes"
description: "Learn how to add location notes and stop descriptions, and understand who can see them across different platforms."

Rider's may have specific instructions about pickup or dropoff that you want to communicate to your agency's drivers. Or perhaps you've configured specific stop locations that you want to communicate information about to riders and drivers. This article answers questions like: 

How do I add a location note? 

Who can see the location note, and where? 

How do I add a stop decsription? 

Who can see the stop description, and where?

## What happens when I add a Location Note?

You can add these notes right under pickup or dropoff locations:

![](https://downloads.intercomcdn.com/i/o/1082098701/b1627f3138a7094633efaf18/1.png?expires=1753820100&signature=26a2a583776fa5ac691dad5becbc5298abb2cc59b63180c8ed98dd91c115c0f0&req=dSAvFMl3lYZfWPMW1HO4zS%2BO6ovatQ8WXHcqN9fRYHO5UXD3423%2FZG%2Fj8cp7%0AEdZ5%0A)Once booked, these notes can be viewed on the request page:

![](https://downloads.intercomcdn.com/i/o/1082098985/cfea262890918a284ff9ef15/2.png?expires=1753820100&signature=1035b55b6c877688726d6b3f13008a3b6dbc8e22bdd780fae0ea615465d287fb&req=dSAvFMl3lYhXXPMW1HO4zTumd04GeY91zVpE8mhpDMTNfxr5iTIf4Wo5RHWQ%0AHg1n%0A)![](https://downloads.intercomcdn.com/i/o/1082099159/54a0d94d6d6958b9e1799840/3.png?expires=1753820100&signature=61fe487df77a696e6b9bb80cbf7826785ee6db4cbc198548a7520f29d9779611&req=dSAvFMl3lIBaUPMW1HO4zSllPyPIyQmtEtCW67thBkke6ruwZPrbGBxhyjqY%0AhSdh%0A)

Then they will be displayed to drivers in the driver app:

![](https://downloads.intercomcdn.com/i/o/1082099394/abaaf186a44d878c48b6ab4c/Screenshot+2024-06-14+at+8_47_49%E2%80%AFAM.png?expires=1753820100&signature=a4a8eba5f7a861b11ee4b7b92737610c36b522e6c29140bdf5179599f982a3c2&req=dSAvFMl3lIJWXfMW1HO4zROncnKr3tYOScg3WLQdH0nO7DSZMUT%2FenxgfrlJ%0AZGp8%0A)

In the rider app, you can choose if you’d like for these notes to be visible to riders or not. Speak to your Spare Representative to toggle this on or off:

![](https://downloads.intercomcdn.com/i/o/1082100020/ea3dade2358140b14ba4eab7/Screenshot+2024-06-14+at+8_48_17%E2%80%AFAM.png?expires=1753820100&signature=983701731f7f5eb2bef118a8e4ba627162b93d1afeb20c359fe9bec7e1c3fddd&req=dSAvFMh%2BnYFdWfMW1HO4zZsiWk1a69Qe86on9WuBMQw7NUOT9sGRrSrFIdci%0A57zo%0A)Along with that same choice, you can choose if riders should be able to add or not add these notes from the rider app.

## What happens when I add a stop description?These can be added when you create or edit a stop. 

You can add a description for riders and for drivers separately:

![](https://downloads.intercomcdn.com/i/o/1082101093/d73a75735d15bc64843a566d/4.png?expires=1753820100&signature=4a2cb8ccd8f5c3e54998a4417d43f477924f19e16039fde3dde7f5c330969805&req=dSAvFMh%2BnIFWWvMW1HO4zZ3jzwQUh%2FOWkM0k6bVKHuTpwzun%2Fid%2B2EJzkkYA%0ALJ0d%0A)For drivers, information for the stop is shown at the top (as there may be multiple riders picked up at the same stop:

![](https://downloads.intercomcdn.com/i/o/1082103132/586c21733963fdff08aa7751/Screenshot+2024-06-14+at+8_50_20%E2%80%AFAM.png?expires=1753820100&signature=86a284d217fe798648305802332dfe2cf6f0b69910797bc3dc619c9d4c79e967&req=dSAvFMh%2BnoBcW%2FMW1HO4zeInKTKDAI%2FQM5RbAPZRB3uWr%2FDoPRhejTl9pimR%0AindG%0A)

For riders, the description for them is visible on the trip timeline:

## Can I get the Pickup or Dropoff Note to always be the same for a certain rider?Yes! By using Favourite Locations, you can have the pickup or dropoff note pre-filled every time a request is made for that location & rider. 

This can be found under preferences for the rider:

![](https://downloads.intercomcdn.com/i/o/1082105695/ed3a8890e83af462b508aca9/8.png?expires=1753820100&signature=c452363e729655269f1f475250379f77f2337e80ab5761076ccda8ec2598826e&req=dSAvFMh%2BmIdWXPMW1HO4zaO5vvOs7wHq30kV1IFFIWTgtuCfGRE8E7Cn2NqD%0AkAZ3%0A)You can add a favourite location, with a note:

![](https://downloads.intercomcdn.com/i/o/1082106109/fa0e627627a5979a1ece2763/9.png?expires=1753820100&signature=38706a623ca1a3695017974ae096a06bcd4bb73f9ad85857ca3b638fb3000381&req=dSAvFMh%2Bm4BfUPMW1HO4zZjAvRG0exgvqld98diaFb6Lh4swXMmZ%2F2tVl70P%0Ay9g7%0A)Then, this will populate every time that rider uses that location:

## Do notes show to external drivers?Yes. All the Spare notes added to a request are passed through to Open Fleets providers. Notes are send in the format: "*`Your Request Note`* **P/U:** *`Your Pickup Location Note`* **D/O:** *`Your Dropoff Location 

Note`*"
