# About Peek View

Peek view is a faster, easier way to view and edit rider information. Instead of navigating to a new page, peek view opens rider details in a side panel on the same page. This means you can easily edit a request, or make changes to a rider's profile, all without leaving the rider page.

# How it works

Like before, you can update rider information by clicking the **Edit** button in any section. However instead of taking you to a new page, a side panel will appear where you can make changes. Complete actions like editing a favorite location, adding memberships, or updating a payment method. Click **Save** and the side panel will close, leaving you right where you left off.

You can even view rider request details this way; click on the request to view details, or add pickup notes. To exit peek view, simply click on the arrow icons in the top left corner or anywhere outside of the panel.
