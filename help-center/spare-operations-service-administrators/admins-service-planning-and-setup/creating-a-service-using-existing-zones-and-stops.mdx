## About this FeatureBy using existing service area and stop data, new services can be set up in minutes! 

This eliminates the need to manually recreate stops for each new service and ensures that updates apply across all relevant services.

[Click here](https://help.sparelabs.com/en/articles/8994903-adding-zones-and-stops) for a guide to creating new zones and stops using the old method.

## ## Creating a Service using Areas and Stops**Create a New 

Service**

- Navigate to the **Service Planning** section.

Click **Add Service**.

**Add a Zone**

- Click **Add Zone** and either select an existing area or click Edit Areas to add or modify an existing zone. 

To use Stops, change the type from **Door to Door** to **Closest Stops**

**Select Stops from the Stop Library**

- Click the **Select Stops** button.

A modal will appear with a list of all available stops in the Stop Library.

Use the search or scroll to choose the stops you need. The search will check both Stop Names and Stop Codes.

Select individual stops or click **Select All** to bulk add stops.

Click **Add Stops** to finalize your selection.

**Review and Save the Service**

- If any of the selected stops fall outside of the Area, you will see an Action Needed error. Update the Area or select new Stops to resolve this.

Click **Save All Changes** to finalize the service.
