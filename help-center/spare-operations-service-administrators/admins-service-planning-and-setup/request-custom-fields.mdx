**Request Custom Fields **are a part of [Custom Fields](https://help.sparelabs.com/en/articles/8994536-custom-fields). They are designed** **to provide** additional information on the trip** while the request is being booked. 

Request custom fields can be seen in:

Spare Operations, where a booking agent can view or edit it

Spare Rider, where a rider can either view or view and edit

Spare Driver, where a driver can either view or view and edit

For the latter two, it can also be configured so that it's not visible.

## How to set up a Request Custom Field?As with any other custom field, you can set Request Custom Fields by going to your **Organization Settings**, selecting **Field Manager,** and navigating to the **

- Request section**.

![](https://spare-74fa64f80617.intercom-attachments-1.com/i/o/972946059/fa6dd3b1b2d311a434099381/K_850kGq1WlssdakDu3OozJ3-YYo4-dKvA.png?expires=1753820100&signature=55a8e2286a2927051411cb3f5c6331d6d4efcec6fa629737244adb962a351fed&req=fSclH814nYRWFb4f3HP0gIKthJao6GZ59RaX40bFHawpG%2B7YxrDdPEXeuKXJ%0Acts%3D%0A)

Click **+** **Add Custom Field **and **fill out the form**. Click **Select Custom Field **to apply changes.

![](https://spare-74fa64f80617.intercom-attachments-1.com/i/o/972946067/a04b518a32f3a01cea8f05ab/qwk8BRnjaw7bo9XaEKTVztiXHKrg3-FlhQ.png?expires=1753820100&signature=3b006f9beae5d96fccfa66f284d691c1b0ffb4f77ef1e191a67d9c1b6591a3a2&req=fSclH814nYdYFb4f3HP0gEUwwtybaN1EdfPYFzxzeRfRnZ7MW%2FtfLBNyjsF6%0AC6k%3D%0A)

Once the field is created, it will be visible in the Request Booking form from the Admin side. It will also become visible in the Rider App.
