Promotional codes are a great way to market your service, build awareness, push riders towards using the rider app, as well as to provide free or discounted trips to unhappy riders. Below is a step-by-step guide on how to create and apply promo codes.

## Creating Your Promo Code

To create a promo code, click on **Spare Pay** in the Spare Operations menu and then click on **Promos**.

![Promo codes interface](https://spare-74fa64f80617.intercom-attachments-1.com/i/o/971509996/469c0520aadb49d2513f7637/Vb_xMhTxRzm0Q2UKQGMejSv5oZr4OrAzoQ.png?expires=1753820100&signature=e1cafcdc9fa7ead8b1f2ee5413ff9fffda40f09c171f915f724431d00b0c6357&req=fScmE8l3lIhZFb4f3HP0gJqK7isJZbObbYTRr1rfYbE89mLsgDVJKkyvdnY6%0ARh4%3D%0A)

There are a few things you will need to configure before you are done creating your promo code:

- **Expiry Time:** The date and time your promo code will expire. After this date and time, riders will not be able to use the promo code anymore, even if they have already applied it to their profile. The promo code will expire and can no longer be used to pay for rides.
- **Secret Code:** The code that will be used to apply the promo to their profile (see below).
- **Description:** The description of the code that will be shown to riders when they enter the code into the rider app.
- **Status:** The status of the code. The code can be Active, Disabled, or Archived.
- **Max Value:** The maximum value of the promo. After the rider has used up the maximum value, the promo code is inactive for that rider and can no longer be used.
- **Max Rides Per User:** The maximum number of trips this promo code can be used towards.
- **Max Users:** The maximum number of riders that can apply the promo code.

## Applying Promo Codes

If you click on a promo code, you can see if anyone has applied it to their profile and how much they've used/redeemed. 

![](https://spare-74fa64f80617.intercom-attachments-1.com/i/o/971510004/eea6a2afa3beda700a7953b6/oCcGcDjIcj_n099yxT1_Mr9IYMomipdGsA.png?expires=1753820100&signature=caca8da9136f3239e906331bf8715d783d1d332fd8e040ffb79adf0c5be39765&req=fScmE8h%2BnYFbFb4f3HP0gD3eqZcVrofGj7cRf7k%2FXKVsuXmLn%2BGQLHMnuMKD%0Arj4%3D%0A)

You can **apply a promo code to a rider's profile** by clicking **Apply Promo** at the top and then finding the rider.

![Apply promo code interface](https://spare-74fa64f80617.intercom-attachments-1.com/i/o/971510007/3b3e47e0273d04cca06e897e/t7Oyu0wycJZVSQh49ejSRnUmTd8XOfS8Cg.png?expires=1753820100&signature=a5d11fa59e5709558a73df8375a5dd2d0791c9eed4984c8badc2d1e146f672c3&req=fScmE8h%2BnYFYFb4f3HP0gAggNN56KXTXdAkwQoXImtFeNZXxPAw5KEYWaAEE%0An78%3D%0A)

You can also apply a promo code right to a rider's profile at the top.
