Welcome to the **Work Order Management** guide, your resource for efficiently managing tasks within Spare EAM.

## What is Work Order Management?

<Info>
Work Order Management in Spare EAM streamlines the creation, tracking, and completion of tasks related to your organization's assets. From scheduling maintenance to resolving issues, this module helps ensure smooth operations and accountability.
</Info>

## Key Features

<CardGroup cols={2}>
  <Card title="Work Order Lifecycle" icon="cycle">
    - **Open**: Create and assign new tasks to individuals or teams
    - **In Progress**: Track ongoing work and update status as needed
    - **Completed**: Mark tasks as finished and record completion details
  </Card>
  <Card title="Detailed Work Records" icon="file-text">
    - Capture task descriptions, assigned personnel, and linked assets or issues
    - Record scheduled and actual completion dates for accurate tracking
  </Card>
  <Card title="Audit and History" icon="history">
    - Maintain clear records of all work orders including who completed them and when
    - Easy access to historical data for reporting and compliance
  </Card>
  <Card title="Seamless Integration" icon="link">
    - Link work orders to assets, issues, and parts for comprehensive task visibility
    - Connect maintenance activities across all system modules
  </Card>
</CardGroup>

## How to Use Work Order Management

<Tabs>
  <Tab title="Creating a Work Order">
    <Steps>
      <Step title="Navigate to Work Orders">
        Go to the **"Work Orders"** section in Spare EAM
      </Step>
      <Step title="Start New Work Order">
        Click on **"New Work Order"**
      </Step>
      <Step title="Fill Required Details">
        Complete the following information:
        - **Description**: Clearly explain the task
        - **Assign To**: Select the individual responsible
        - **Schedule Date**: Specify when the task should start and be completed
        - **Linked Items**: Attach relevant assets, issues, or parts
      </Step>
      <Step title="Save and Activate">
        Save the work order to activate it
      </Step>
    </Steps>
  </Tab>
  
  <Tab title="Updating a Work Order">
    <Steps>
      <Step title="Open Work Order">
        Open the specific work order from the list
      </Step>
      <Step title="Modify Details">
        Update details such as status, assigned personnel, or linked items
      </Step>
      <Step title="Save Changes">
        Save the changes to keep the record up to date
      </Step>
    </Steps>
  </Tab>
  
  <Tab title="Completing a Work Order">
    <Steps>
      <Step title="Open Work Order">
        Once the task is finished, open the work order
      </Step>
      <Step title="Mark as Completed">
        Mark the work order as **Completed**
      </Step>
      <Step title="Add Completion Details">
        Add the actual completion date and any relevant notes
      </Step>
      <Step title="Finalize">
        Save the work order to finalize it
      </Step>
    </Steps>
  </Tab>
</Tabs>

<Tip>
Use the work order status updates to keep all team members informed about task progress and maintain accurate project timelines.
</Tip>
